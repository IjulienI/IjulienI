### 🧭 About

I'm a 4th-year **Game Programming** student at **ArtFX Montpellier**, specialized in **C++ / Unreal Engine 5** gameplay systems, with a serious interest in the layer *under* the gameplay: engine architecture, custom allocators, and editor tooling.

I like building things that sit between the engine and the designers. Plugins, custom components, analytics pipelines, reusable gameplay frameworks.
And because gameplay is only as good as the loop that runs it, I spend time reading Mike Acton and Jason Gregory on the side.
